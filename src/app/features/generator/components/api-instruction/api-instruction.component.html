PESEL Generator API

You can generate valid PESEL numbers not only in the browser UI but also via a simple HTTP API.

This is useful for:

  • automated testing
  • generating test users
  • QA environments
  • scripting
  • integration with other systems

Base endpoint:

        GET https://api.pesel.dev/generator

Query Parameters

All parameters are optional.

Parameter Type Description
quantity number Number of PESEL numbers to generate (1–10000). Default: 1
sex string Filter by sex: male or female
dob string Date of birth in format dd.MM.yyyy
